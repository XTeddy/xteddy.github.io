1. 块级元素

1.1 标题

① 两个或两个以上 = 字符显示为一级标题：

这是一级标题
==

显示效果：

② 两个或两个以上 - 显示为二级标题：

这是二级标题
--

显示效果：

③ 1~6级标题可用 # 个数对应：

\# 一级标题
\## 两级标题
\### 三级标题
\#### 四级标题
...

显示效果：

1.2 强制换行

行尾添加至少两个空格+回车，则段落被强制换行。

1.3 块引用

① > 符号表示块引用：

\> 这是一个块引用吗？
\> 是的！

显示效果：

这是一个块引用吗？
是的！

1.4 列表（有序和无序）

① 有序列表：阿拉伯数字+英文句号.+空格：

1. 有序列表1
2. 有序列表2
3. 有序列表3

显示效果：

1. 有序列表1
2. 有序列表2
3. 有序列表3

② 无序列表：*，连字符-或+号 + 空格：

\* 无序列表
\- 无序列表
\+ 无序列表

显示效果：

• 无序列表
• 无序列表
• 无序列表

2. 内联元素

2.1 链接方式（内联和引用）

① 内联：

这是我的博客地址：\[点击访问](https://bdongs.com "bdongs")。
这是我的博客地址：<https://bdongs.com/>

显示效果：

这是我的博客地址：点击访问。 这是我的博客地址：https://bdongs.com/

② 引用：

这是我的博客地址：[点击访问][id]
[id]: https://bdongs.com //可在文章任意位置且单独占一行

这是我的博客地址：[点击访问][]
[点击访问]: https://bdongs.com //可在文章任意位置且单独占一行

显示效果：

这是我的博客地址：[点击访问][id]
[id]: https://bdongs.com

这是我的博客地址：[点击访问][]
[点击访问]: https://bdongs.com

2.2 强调

① 斜体：

*这是斜体*

显示效果：

这是斜体

②粗体：

**这是粗体**

显示效果：

这是粗体

2.3 代码

①多个`:

```
#include <stdio.h>
int main()
{

printf(“Hello, World! \n”);

return 0;

}

```

显示效果：

②单个`:

`printf(“Hello, World! \n”);`

显示效果：
printf("Hello, World! \n");
